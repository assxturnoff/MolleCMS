## Wymagania

- Serwer WWW lub VPS
- PHP 7 lub wyższa wersja

## Instalacja
- Pobierz lub sklonuj repozytorium
- Umieść pliki na serwerze
- Stwórz bazę danych importując schemat bazy z pliku hzpriv.sql
- Plik konfiguracyjny znajdziesz w katalogu "readme.md"
